How to use

1

In QBO, go to Banking → Rules and export the client's rules as an .xls file.

2

Drop that file here (or click to browse). Every rule gets - [PA] as the memo — anything else gets overwritten.

3

Download the processed file, then reimport via Banking → Rules → Import.

⚠ Before importing: go to Settings → Advanced → Chart of Accounts and temporarily turn off account numbers. Re-enable after the import. This ensures QBO matches categories correctly.
